This plugin show visited pages, post or any custom post type tracking in user profile.

== Description ==

This plugin helps to track the user activity over the website. Admin can check the user's visited pages, posts, or any custom post type pages in the admin panel. Admin can view the tracking once click on the edit the user profile page.

== Installation ==

1. Upload the entire `user-page-post-tracking` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
